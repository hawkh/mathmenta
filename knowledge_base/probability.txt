# Probability and Statistics for JEE

## Basic Probability

### Fundamental Concepts:
- Sample Space (S): Set of all possible outcomes
- Event (E): Subset of sample space
- Probability: P(E) = n(E)/n(S)

### Axioms of Probability:
1. 0 ≤ P(E) ≤ 1
2. P(S) = 1
3. For mutually exclusive events: P(A ∪ B) = P(A) + P(B)

### Complement:
P(E') = 1 - P(E)

### Addition Theorem:
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

### Conditional Probability:
P(A|B) = P(A ∩ B)/P(B) where P(B) ≠ 0

### Multiplication Theorem:
P(A ∩ B) = P(A)·P(B|A) = P(B)·P(A|B)

### Independent Events:
A and B are independent if:
- P(A ∩ B) = P(A)·P(B)
- P(A|B) = P(A)
- P(B|A) = P(B)

## Bayes' Theorem

### Total Probability:
If B₁, B₂, ..., Bₙ are mutually exclusive and exhaustive:
P(A) = Σ P(Bᵢ)·P(A|Bᵢ)

### Bayes' Formula:
P(Bᵢ|A) = P(Bᵢ)·P(A|Bᵢ) / Σ P(Bⱼ)·P(A|Bⱼ)

### Applications:
- Medical testing
- Quality control
- Decision making under uncertainty

## Random Variables

### Discrete Random Variables:
- Takes countable values
- Probability Mass Function (PMF): P(X = x)
- Σ P(X = x) = 1

### Continuous Random Variables:
- Takes uncountable values in an interval
- Probability Density Function (PDF): f(x)
- P(a ≤ X ≤ b) = ∫[a to b] f(x)dx
- ∫f(x)dx over all x = 1

### Cumulative Distribution Function (CDF):
F(x) = P(X ≤ x)
- For discrete: F(x) = Σ P(X ≤ x)
- For continuous: F(x) = ∫[-∞ to x] f(t)dt

## Mathematical Expectation

### Mean (Expected Value):
- Discrete: E(X) = Σ x·P(X = x)
- Continuous: E(X) = ∫ x·f(x)dx

### Variance:
Var(X) = E[(X - μ)²] = E(X²) - [E(X)]²

### Standard Deviation:
σ = √Var(X)

### Properties:
- E(aX + b) = aE(X) + b
- Var(aX + b) = a²Var(X)
- E(X + Y) = E(X) + E(Y)

## Important Probability Distributions

### Discrete Distributions:

#### Binomial Distribution:
- Parameters: n (trials), p (success probability)
- PMF: P(X = k) = C(n,k) p^k (1-p)^(n-k)
- Mean: μ = np
- Variance: σ² = np(1-p)
- Conditions: Fixed trials, independent, constant p

#### Poisson Distribution:
- Parameter: λ (average rate)
- PMF: P(X = k) = (λ^k e^(-λ))/k!
- Mean: μ = λ
- Variance: σ² = λ
- Used for: Rare events, arrivals, defects

### Continuous Distributions:

#### Uniform Distribution:
- PDF: f(x) = 1/(b-a) for a ≤ x ≤ b
- Mean: μ = (a+b)/2
- Variance: σ² = (b-a)²/12

#### Normal (Gaussian) Distribution:
- PDF: f(x) = (1/σ√(2π)) e^(-(x-μ)²/(2σ²))
- Mean: μ
- Variance: σ²
- Standard Normal: Z = (X - μ)/σ ~ N(0,1)
- Properties: Symmetric, bell-shaped

#### Exponential Distribution:
- PDF: f(x) = λe^(-λx) for x ≥ 0
- Mean: μ = 1/λ
- Variance: σ² = 1/λ²
- Used for: Waiting times, lifetimes

## Statistics

### Measures of Central Tendency:

#### Mean:
- Arithmetic: x̄ = Σxᵢ/n
- Weighted: x̄_w = Σwᵢxᵢ/Σwᵢ

#### Median:
- Middle value when arranged in order
- For n odd: (n+1)/2 th term
- For n even: Average of n/2 and (n/2+1) th terms

#### Mode:
- Most frequent value

### Measures of Dispersion:

#### Range:
Range = Maximum - Minimum

#### Mean Deviation:
MD = Σ|xᵢ - x̄|/n

#### Standard Deviation:
s = √[Σ(xᵢ - x̄)²/n]

#### Variance:
s² = Σ(xᵢ - x̄)²/n

#### Coefficient of Variation:
CV = (s/x̄) × 100%

## Correlation and Regression

### Correlation Coefficient:
r = Σ(xᵢ - x̄)(yᵢ - ȳ) / √[Σ(xᵢ - x̄)² · Σ(yᵢ - ȳ)²]

Properties:
- -1 ≤ r ≤ 1
- r = 1: Perfect positive correlation
- r = -1: Perfect negative correlation
- r = 0: No linear correlation

### Regression Lines:

#### Y on X:
y - ȳ = b_yx(x - x̄)
where b_yx = r·(s_y/s_x) = regression coefficient

#### X on Y:
x - x̄ = b_xy(y - ȳ)
where b_xy = r·(s_x/s_y)

### Properties:
- b_yx · b_xy = r²
- Both lines pass through (x̄, ȳ)
- Angle between lines: tan θ = [(1-r²)/r] · [s_x s_y/(s_x² + s_y²)]

## Problem-Solving Strategies:
1. Clearly identify sample space and events
2. Check for independence before multiplying probabilities
3. Use tree diagrams for sequential events
4. Apply Bayes' theorem for inverse probabilities
5. Verify that probabilities sum to 1
6. For distributions, identify parameters first
7. Use standard normal tables for normal distribution
